backend API endpoint documentation


ini adalah server API buat lempar tangkap data dari database
server frontend dipisah ya guys, ambil data disini pake javascript fetch()
front-end bisa ambil data dengan end-point berikut:




    
CHECK APAKAH USER LOGIN ATAU BELUM (WAJIB AUTH DISETIAP PAGE USER)

    https://fashionid-production.up.railway.app/auth
    key: none                   result: {
                                    message -> pesan berhasil atau error     
                                    err -> bernila 1 jika bermasalah
                                }






    
SUBMIT DATA REGISTER

    https://fashionid-production.up.railway.app/user/register     
    key: {                      result: {
        nama,                       message,
        username,                   err
        password,               }
        email,
        telepon
    }






    
USER LOGIN

    https://fashionid-production.up.railway.app/user/login     
    key: {                      result: {
        username,                   message,
        password                    err
    }                           }






    
LOGOUT

    https://fashionid-production.up.railway.app/user/logout     
    key: none                   result: {
                                    message,
                                    err
                                }






    
UPDATE DATA USER

    https://fashionid-production.up.railway.app/user/update     
    key: {                      result: {
        nama,                       message,
        username,                   err
        telepon,                }
        email,
        picture
    (key tergantung data yang di perbarui)
    }
