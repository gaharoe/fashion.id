backend API endpoint documentation



ini adalah server API buat lempar tangkap data dari database
server frontend dipisah ya guys, ambil data disini pake javascript fetch()
front-end bisa ambil data dengan end-point berikut:

    >   https://fashionid-production.up.railway.app/auth -> check user login atau belum
            key: none                   result: {
                                            message -> pesan berhasil atau error
                                            err -> bernila 1 jika bermasalah
                                        }
        
    >   https://fashionid-production.up.railway.app/user/register -> submit data register
            key: {                      result: {
                nama,                       message,
                username,                   err
                password,               }
                email,
                telepon
            }
    
    >   https://fashionid-production.up.railway.app/user/login -> submit data login
            key: {                      result: {
                username,                   message,
                password                    err
            }                           }
            
    >   https://fashionid-production.up.railway.app/user/logout -> logout user
            key: none                   result: {
                                            message,
                                            err
                                        }

    >   https://fashionid-production.up.railway.app/user/update -> update user
            key: {                      result: {
                nama,                       message,
                username,                   err
                telepon,                }
                email,
                picture
            (key tergantung data yang di perbarui)
        }                           
                              
    >   https://fashionid-production.up.railway.app/user/profile -> get user
            key: {                      result: {
                username,                   username,
            }                               nama,
                                            telepon,
                                            email
                                        }
        
    >   https://fashionid-production.up.railway.app/user/delete -> delete user (opsional API)
            key: {                      result: {
                username,                   message,
                password                    err
            }                           }

    >   https://fashionid-production.up.railway.app/rating/create -> tambah rating
            key: {                      result: {
                username,                   message,
                content,                    err
                rating -> 1-5           }
            }                           

    >   https://fashionid-production.up.railway.app/rating/dump -> ambil semua rating
            key: none                   result: {
                                            array: {
                                                username,
                                                content,
                                                rating
                                            }
                                        }

    >   https://fashionid-production.up.railway.app/rating/latest -> ambil rating terbaru
            key: none                   result: {
                                            array: {
                                                username,
                                                content,
                                                rating
                                            }
                                        }
